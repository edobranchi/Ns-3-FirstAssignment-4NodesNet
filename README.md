# Ns-3-FirstAssignment-4NodesNet
First Assignment for the "Network Applications" exam.

Assignment Description

Create a simulation of a network with the following characteristics:

    At least 4 nodes
    Node 0 communicates with node 1
    Node 2 communicates with node 3

Ensure it is possible to collect some data related to the communication between the nodes, such as:

    RTT (Round-Trip Time)
    Throughput
    Packet loss
    TCP parameters (if TCP is used)
    Other metrics

Provide the following:

    The code, preferably through a link to a GitLab repository (in this case, indicate the name of the script and its path).
    A one-page report with the obtained data, preferably visualized with a graph or a table.
